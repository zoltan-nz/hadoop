THE HADOOP EXPERIENCE
=====================

Install Hadoop on Mac:

    $ brew install hadoop
    
Optional:
    
    $ brew install java
    $ brew install gradle

Optional environment settings

    export JAVA_HOME="$(/usr/libexec/java_home)"

* Check you have access to your localhost with `ssh localhost`. Otherwise setup ssh keys.
